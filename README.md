What is this project?
This is a simple program that acts like a school report card. I built it to show how Python can take user data, do math, and give a result.

How it works:
User Input: It asks you to type your marks for 6 subjects (Math, Physics, CS, English, Urdu, and Pak-Studies).

Math Logic: It adds all the marks together and calculates the percentage out of 600.

The Decision Maker: It uses if/elif/else logic to check your percentage and give you a grade like A-1, A, or B.

Why I used these tools:
float(): So the user can enter marks with decimals (like 40.5).

Underscores (_): I learned that Python variable names like pak_studies need underscores because dashes break the code!

if/else: This helps the program "think" and decide which grade to show.
